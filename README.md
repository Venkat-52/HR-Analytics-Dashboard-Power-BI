# HR-Analytics-Dashboard-Power-BI
Interactive HR Analytics Dashboard built using Power BI to analyze workforce metrics, employee attrition, performance, engagement, satisfaction, and training effectiveness through data-driven insights.

## Project Overview

This project presents an interactive HR Analytics Dashboard developed using Power BI to analyze workforce trends, employee attrition, performance metrics, employee engagement, satisfaction levels, and training effectiveness.

The dashboard transforms raw HR data into actionable insights through data cleaning, DAX calculations, KPI tracking, and interactive visualizations.

The objective of this project is to support HR decision-making by identifying workforce patterns, attrition drivers, performance trends, and employee development opportunities.

## Tools & Technologies

- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel
- Data Visualization
- HR Analytics

## Dataset Information

The dataset contains approximately 3,000 employee records including:

- Employee Information
- Department Details
- Employee Status
- Gender
- Age
- Performance Ratings
- Satisfaction Scores
- Engagement Scores
- Training Information
- Attrition Data
- Geographic Information

## Key Insights

- Production department recorded the highest employee attrition.
- Employees aged 51+ showed the highest attrition rate.
- Workforce distribution is highly concentrated in Massachusetts.
- Most employees achieved the "Fully Meets" performance rating.
- Internal and External training participation is nearly balanced.
- Employee engagement and satisfaction demonstrate a positive relationship.
